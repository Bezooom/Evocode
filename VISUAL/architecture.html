🏗️ Ключевые компоненты

🧠 Inference Engine

Дву-модельный локальный инференс на базе llama.cpp

  • FIM-модель (1.5B, <100ms) — догадки кода
  • Chat-модель (8B) — диалог и рассуждения
  • Фоновый режим (background mode)
  • Конфигурируемые промпты

🔀 Умный роутер

Интеллектуальное переключение между локальным и облачным выводом

  • Простые задачи → локальная модель
  • Сложные задачи → OpenRouter (Claude/GPT)
  • Настройка приоритетов и триггеров
  • Контроль расходов (cloud spending)

🔄 Skill Sync Engine

Ежедневная автоактуализация навыков из GitHub

  • Синхронизация каждые 24 часа (03:00)
  • Новые / изменённые / удалённые навыки
  • Soft-delete и бэкапы
  • Верификация перед применением

🛡️ DLP Guard

Защита данных перед отправкой в облако

  • Маскировка API-ключей, токенов, паролей
  • Конфигурируемые правила (regex)
  • Предпросмотр до отправки
  • Логирование заблокированных данных

🔍 Local RAG

Локальный семантический поиск по документам и коду

  • SQLite-vec / FTS5 — без внешних зависимостей
  • Индексация документации и кода проекта
  • Гибридный поиск (векторный + текстовый)
  • Авто-обновление при изменении файлов

🔌 MCP Host

Встроенный хост для Model Context Protocol

  • Серверы: filesystem, sequentialthinking, Gmail
  • Аутентификация и ограничения
  • Мониторинг и логирование вызовов
  • Файрвол для защиты данных

📁 Структура навыков

Системные навыки обновляются автоматически, пользовательские — нет
