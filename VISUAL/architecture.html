Слой интерфейса (IDE)

  • • Брендированная сборка VSCodium без телеметрии Microsoft.
  • • Локальное расширение evocode-agent, управляющее сессиями.
  • • Интегрированные боковые панели мониторинга и настройки Core.

Privacy-платформа Core

  • • Smart Router: анализирует сложность запроса для локального/облачного вывода.
  • • DLP Guard: 6 regex-правил для удаления паролей, JWT и API-ключей.
  • • Локальный RAG на SQLite-vec для расширения контекста.

Локальный инференс

  • • Движок локального исполнения llama.cpp (shared memory).
  • • Выделенный FIM-канал (:8082) для автодополнения кода.
  • • Чат-канал (:8080) для сложных инструкций и выполнения агентов.
