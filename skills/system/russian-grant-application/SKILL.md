# Заявки на гранты РФ

Полный цикл генерации заявок на гранты РФ.

## Целевые программы

- **РНФ** — до 150 млн руб (научные исследования)
- **Старт** — до 3 млн руб (стартапы до 2 лет)
- **Развитие** — до 20 млн руб (стартапы 2-5 лет)
- **Коммерциализация** — до 15 млн руб (готовый продукт)
- **Сколково** — льготы + инвестиции (резиденты)

## Workflow

1. Выбор программы → анализ соответствия критериям
2. Описание проекта (актуальность, новизна, TRL)
3. Коммерческий потенциал (рынок, конкуренты, бизнес-модель)
4. Команда проекта
5. Бюджет и календарный план
6. Генерация презентации для защиты
