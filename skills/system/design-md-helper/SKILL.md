# DESIGN.md Helper - Smart Style Assistant

## Когда использовать
- Пользователь пишет: "сделай стильно", "как у крутых", "современно", "дорого"
- Запрос размытый: "нужен красивый дашборд/лендинг/форма"
- Пользователь открыт к советам: "что посоветуешь?", "какой стиль подойдёт?"

## Логика работы агента

### 1. Конкретный запрос ("в стиле Linear")
Открой references/{brand}/DESIGN.md и следуй токенам строго.

### 2. Размытый запрос ("стильно", "как у крутых")
- Дашборд/админка → предложи: linear, superhuman, raycast
- Лендинг/маркетинг → предложи: vercel, stripe, framer
- Документация/блог → предложи: nextjs, tailwind, shadcn
- Минимализм/мобайл → предложи: apple, arc, notion

Предложи 2-3 варианта с описанием вайба. После выбора пользователя открой соответствующий DESIGN.md.

### 3. Просмотр всех вариантов
Выведи список папок из references/ с короткими дескрипторами:
linear (тёмный техно), vercel (светлый минимализм), stripe (корпоративный чистый), framer (креативный с анимациями), raycast (компактный), superhuman (email-эстетика), claude (дружелюбный светлый) и др.

## Токены: что извлекать
- Цвета: --primary, --background, --text, --border, --accent
- Типографика: font-family, font-size, line-height, letter-spacing
- Компоненты: padding, border-radius, hover/active-состояния
- Эффекты: box-shadow, backdrop-filter, transition, transform

⛔ НЕ выдумывай значения. Используй только то, что есть в DESIGN.md. Если токена нет — спроси пользователя или возьми безопасный дефолт.

## Пример диалога
Пользователь: Сделай форму входа, но чтобы выглядело дорого
Агент: Для дорогого вида рекомендую:
1. Stripe - чистый белый, тонкие границы, акцент на контент
2. Linear - тёмная тема с фиолетовым свечением, техно-люкс
3. Framer - креативные градиенты и микро-анимации
Какой вайб берём?
Пользователь: Бери Linear
Агент: [открывает references/linear/DESIGN.md, берёт --background: #0D0D0D, --primary: #5E6AD2, шрифт Inter, радиус 8px и генерирует код]

## Обновление коллекции
Новые системы добавляются в references/. Просмотр: ls ~/.kilo/skills/design-md-helper/references/
