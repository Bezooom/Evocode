# Adaptive Learning (Адаптивное обучение)

Этот модуль отвечает за динамическую перестройку сложности курса в зависимости от реакций (гипотетических или реальных) ученика.

## 1. Branching Paths (Сюжетно-методическое ветвление)
Если в `LearningPath` ученик плохо справляется с определенной темой (например, не может пройти Босса по теме "Дроби"):
- Система не просто говорит "Неверно", а открывает **Extra Lesson** (Дополнительный уровень).
- Дополнительный уровень объясняет концепт с другой визуальной метафорой или через более простые шаги.

## 2. Dynamic Difficulty Adjustment (Динамическая сложность)
Использование поля `difficulty_micro` (1–5) внутри `LessonBlock`:
- Уровень 1-2: Прямая подсказка (Guided Practice).
- Уровень 3: Стандартная игровая задача.
- Уровень 4-5: Открытая задача на Синтез/Оценку (بدون подсказок, таймер).
Агент должен уметь повышать или понижать этот уровень в зависимости от предыдущих выборов ученика в интерактивной презентации.

## 3. Micro-adaptation (Внутри LessonBlock)
- В `AssessmentQuiz` и `BossBattleMechanic` прописывать механику "Второй шанс": если ученик ошибается, следующая опция/вопрос дает скрытую подсказку.
- Геймификация поражений (Fail States): поражение не должно расстраивать. Это шанс получить новый "предмет" для изучения.
