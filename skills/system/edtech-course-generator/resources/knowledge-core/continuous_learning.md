# Sovereign Knowledge & Continuous Learning

Экосистема "ЭдТех-Комбайн Level 8" является суверенной (Sovereign). Она не только работает локально, но и обладает собственной, самоактуализируемой базой знаний.

## 1. Актуализация Фреймворков (Weekly Curated RAG)
Раз в неделю Агент автоматически формирует поисковые запросы по базам научных публикаций (Google Scholar, PubMed, Semantic Scholar) по темам:
- `Cognitive Load Theory updates`
- `Self-Determination Theory in gamification`
- `Attention span metrics in Gen Alpha`

Найденные абстракты прогоняются через встроенный RAG, и если появляется научно обоснованный консенсус (например, "Оптимальная длина микрообучения для 10 лет снизилась с 12 до 9 минут"), Агент обновляет файлы в `resources/age-profiles/`.

## 2. Confidence Scoring (Оценка достоверности)
Каждый факт, загружаемый в `GraphRAGSummary`, теперь должен иметь `confidence_score` (0.0 - 1.0) и `timestamp`.
- Если факт взят из старого учебника (до 2020 года) и касается технологий, `confidence_score` понижается.
- Агент помечает такие узлы в LTKG как `needs_verification` и предупреждает учителя в `Teacher Dashboard`.

## 3. Суверенитет Данных
Все извлеченные мета-данные хранятся строго в локальных `.md` и SQLite/VectorDB базах. Система не зависит от закрытия внешних API или цензуры сторонних провайдеров.
