# Autonomous Curriculum Planner

Автономное планирование многолетнего трека обучения (Multi-Year Learning Path). На 5 уровне навыка Агент умеет выходить за рамки одного курса и строить долгосрочные стратегии.

## 1. Multi-Year Learning Path
Вместо изолированного генератора курсов, Агент теперь проектирует 6–12-месячные дорожные карты:
- **Macro-Goals:** Годовые образовательные цели (например, "От 0 до разработки своей первой 2D-игры").
- **Milestones:** Промежуточные курсы, каждый со своей Narrative Arc.
- **Dependency Mapping:** Строгое отслеживание того, какие курсы должны предшествовать другим.

## 2. Автоматическое выявление Skill Gaps (Пробелы в знаниях)
На основе `course_history.json` и `Long-Term Knowledge Graph`:
1. Агент сканирует результаты прохождения прошлых курсов (особенно `struggled_with` и Fail States).
2. Выявляет систематические просадки (например, ребенок отлично понимает логику циклов, но проваливает задачи на математические операторы).
3. **Bridge Courses (Мосты):** Агент АВТОМАТИЧЕСКИ предлагает и генерирует короткий "мост-курс" (на 1-2 урока), который в игровой форме закрывает выявленный пробел перед переходом к следующему большому Milestone.

## 3. Оркестрация планировщика
- Когда пользователь запрашивает новый курс, `Curriculum Planner` сперва валидирует: "Достаточно ли у профиля X пререквизитов для этой темы?".
- Если нет — он отвечает `LearningPath` с внедренными пререквизитными блоками-введениями, адаптированными под любимый стиль геймификации ребенка.
