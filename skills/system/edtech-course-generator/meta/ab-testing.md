# A/B Testing & Self-Improvement

## Логика проведения A/B тестов
Система `edtech-course-generator` может генерировать две версии одного урока для тестирования гипотез вовлеченности (Engagement).

### Как настроить сплит:
1. **Version A (Контрольная):** Стандартный набор механик (например, 5E Model).
2. **Version B (Экспериментальная):** Изменение одной модальности. Например:
   - Замена голоса в Kokoro TTS с нейтрального на гипер-экспрессивный.
   - Изменение стиля ComfyUI (Pixar 3D vs Cyberpunk 2D).
   - Разные геймификационные фреймворки (Сбор артефактов vs Boss Battle).

## Auto-Analysis (Автоматический анализ)
Каждые 5–10 генераций Агент Оркестратор читает `meta/performance-log.md` и выводит Summary:
- Сравнение `Critic Score` для разных подходов.
- Анализ `User Feedback`.
- **Actionable Insight:** "Использование Sci-Fi сеттинга для 10-12 лет дает +15% к Critic Score по вовлеченности. Рекомендуется обновить `gamification_patterns.md`."

Все успешные инсайты автоматически переносятся в `meta/prompt-evolution.md`.
