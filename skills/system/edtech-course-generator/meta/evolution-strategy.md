# Self-Evolution Engine (Движок Саморазвития)

Агент 5-го уровня автономен в улучшении собственных системных промптов и шаблонов.

## 1. Evolutionary Prompt Optimization (EPO)
Когда Агент получает стабильно низкие оценки (от Критика или `User Feedback`), он инициирует алгоритм **EPO (Evolutionary Prompt Optimization)**:
1. **Mutation:** Агент генерирует 3–5 альтернативных версий своего системного промпта (например, меняет тон подкаста с "комедийного" на "документальный", или меняет `image_prompt` стиль).
2. **A/B/C/D Testing:** Запускает скрытые тесты для всех вариантов через `Learner Simulation Critic`.
3. **Selection:** Вариант с максимальным `Predicted Engagement Score` заменяет текущий бейслайн.
4. **Logging:** Запись изменений в `meta/prompt-evolution.md` с гипотезой и результатом.

## 3. Авто-наполнение Golden Examples
- Любой сгенерированный курс, который получает `Final Score >= 98` от Critic Agent и `User Feedback >= 4.8/5.0`, АВТОМАТИЧЕСКИ сохраняется в папку `resources/golden-examples/`.
- Файл очищается от PII (персональных данных) и становится новым Few-Shot эталоном для LLM на последующие запуски. Это создает эффект компаунда качества.
