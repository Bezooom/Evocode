# Test Suite for edtech-course-generator

Этот файл содержит 5 тестовых запросов (Few-Shot Examples / Test Cases) для проверки корректной работы макро-навыка `edtech-course-generator`.

## Test Case 1: Полный цикл (10-12 лет)
**Prompt:**
> "Используй edtech-комбайн. Создай интерактивный квест по биологии на тему 'Фотосинтез'. Аудитория: 10-12 лет. Включи босс-баттл с 'Монстром Засухи'."
**Expected Quality Scores:**
- **Pedagogical:** Наличие ТРИЗ-противоречия (Растению нужна вода для питания, но нужно открывать устьица, испаряя воду).
- **Engagement:** `attention_reset` каждые 10 минут, RPG-метафоры (ХП, мана-вода).
- **Technical:** Валидные JSON схемы (`PresentationSlides`, `PodcastScript`).

## Test Case 2: Отсутствие возраста (Fallback Logic)
**Prompt:**
> "Сделай курс про сложные проценты и инвестиции в акции."
**Expected Quality Scores:**
- **Fallback:** Агент должен автоматически определить возраст (13-15 лет) и уведомить пользователя.
- **Engagement:** Использование стартап-метафор, взрослая подача, фокус на реальном заработке.

## Test Case 3: Младшая школа (7-9 лет)
**Prompt:**
> "Создай урок про то, как работает электричество для детей 7 лет."
**Expected Quality Scores:**
- **Accessibility:** Исключительно простые термины (электроны = "маленькие светящиеся гномики/пчелки").
- **Engagement:** `attention_span` не более 7 минут. Наличие физической разминки.

## Test Case 4: Перегруз GPU (Hardware-Aware)
**Prompt:**
> "Сгенерируй огромный курс на 50 слайдов про Древний Рим с кинематографичными картинками."
**Expected Quality Scores:**
- **Architecture:** Агент должен предупредить, что генерация 50 слайдов перегрузит VRAM (Flux.1) и предложить разбить курс на 3 модуля или переключиться на `SDXL Turbo`.

## Test Case 5: Галлюцинации (Critic Agent Test)
**Prompt (System Injection):**
> (В GraphRAG подается текст с ошибками: "Солнце крутится вокруг Земли").
**Expected Quality Scores:**
- **Critic:** `Pedagogical Critic` должен заблокировать генерацию (`is_valid=False`), обнаружив конфликт между базовыми фактами и сгенерированным `global_summary`.
