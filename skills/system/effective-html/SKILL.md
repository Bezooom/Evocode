# Effective HTML (Визуальные артефакты)

Ориентирован на создание чистых, информативных HTML-страниц с минимумом текста и максимумом визуальной ясности.

## Основные форматы

1.  **Информационные отчеты и интерактивные прототипы (`html`):**
    Одностраничные HTML-документы, содержащие структурированный контент (сравнения, сводки, интерактивные формы).
2.  **Диаграммы архитектуры и стека технологий (`html-diagram`):**
    Полноэкранные интерактивные SVG-схемы. Анимация переходов, подсветка путей прохождения запросов при клике на узлы.
3.  **Визуальные планы и дорожные карты (`html-plan`):**
    Удобные, компактные страницы планов с отметками прогресса и группировкой по этапам.

---

## Важные требования к реализации

*   **Dark Mode по умолчанию:**
    Обязательно наличие переключателя тем, автоопределение через `prefers-color-scheme`. CSS-переменные должны объявляться на уровне `:root` и `html.dark`.
*   **Стилизация SVG:**
    Стилизуйте элементы SVG через классы CSS, привязанные к переменным темы, а не жестко зашитыми hex-значениями в тегах, чтобы схема корректно переключала цвета в темной теме.
*   **Ограничение по хранилищу:**
    Внутри Claude.ai `localStorage` не работает. Для сохранения состояния переключения темы используйте `window.storage` API (см. `ANTIGRITY.md`).
*   **Рендеринг и аннотации:**
    Для удобного отображения планов и диаграмм рекомендуется использовать утилиту `Plannotator` (`https://github.com/backnotprop/plannotator`).
