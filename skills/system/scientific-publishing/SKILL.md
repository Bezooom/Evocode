# Scientific Publishing — Координатор Nature-cluster

Это макро-навык-**навигатор**. Он не выполняет работу сам, а показывает агенту, какой из
7 специализированных навыков загрузить под конкретную подзадачу научной публикации.

## Когда загружать этот навык

Когда запрос пользователя **общий** или **неоднозначный** в контексте научной публикации:

- «Помоги со статьёй»
- «Подготовь материал для Nature»
- «Journal club в пятницу, нужна презентация и анализ статьи»
- «Рукопись для Cell, все вопросы»

Если запрос **конкретный** (например «сделай рисунок 3 в стиле Nature») — сразу
загружай соответствующий специализированный навык, минуя этот координатор.

## 7 специалистов Nature-cluster

| Подзадача | Специалист | Что делает |
|-----------|-----------|-----------|
| Добавить строгие цитирования Nature/CNS к тексту | **`nature-citation`** | Сегментация абзаца, поиск только во флагманах Nature/Science/Cell, экспорт EndNote/RIS/Zotero RDF |
| Построить submission-grade рисунки | **`nature-figure`** | Python (matplotlib/seaborn) или R (ggplot2/patchwork), SVG/PDF/TIFF-экспорт, multi-panel |
| Полировка академического английского до Nature-уровня | **`nature-polishing`** | Реструктуризация прозы по Scientific English Writing принципам + Academic Phrasebank |
| Двуязычный ридер статьи (EN + RU или EN + CN) | **`nature-reader`** | Markdown-ридер с сохранёнными якорями на источник, figure-aware |
| Ответ рецензентам (point-by-point rebuttal) | **`nature-response`** | Аудит и составление response letter на major/minor revision |
| Презентация из статьи (русская) | **`paper2ppt-ru`** | PPTX для защиты диссертации, научного семинара, конференции в РФ. ГОСТ Р 7.0.11-2011 |
| Презентация из статьи (китайская, legacy) | **`nature-paper2ppt`** | Устаревший предшественник `paper2ppt-ru`. Использовать только если нужен именно CN-контент |

## Правило выбора

```
Запрос пользователя → анализ → какой из 7 навыков?

1. «цитаты», «библиография», «найди references» → nature-citation
2. «рисунок», «график», «multi-panel figure», «SVG для Nature» → nature-figure
3. «полировка», «улучши английский», «academic style» → nature-polishing
4. «двуязычный перевод», «читалка статьи», «анкорный ридер» → nature-reader
5. «ответ рецензентам», «rebuttal letter», «response» → nature-response
6. «презентация» + русский/защита/РФ → paper2ppt-ru
7. «презентация» + китайский → nature-paper2ppt (legacy)
8. Общий/неоднозначный → этот координатор (scientific-publishing)
```

## Примеры использования

### Пример 1: Сквозной workflow от PDF к публикации

```
Пользователь: «Помоги подготовить статью для Nature Methods из этого препринта»
Агент:
  1. Загружает scientific-publishing (этот файл) — общий контекст
  2. Декомпозирует задачу на подзадачи:
     - Улучшить прозу → nature-polishing
     - Проверить/добавить цитаты → nature-citation
     - Пересобрать рисунки → nature-figure
  3. По каждой подзадаче загружает нужный специалист и выполняет
  4. Финальная сборка
```

### Пример 2: Одна подзадача

```
Пользователь: «Отполируй introduction для Nature»
Агент:
  1. Распознал: одна подзадача — полировка текста
  2. Сразу загружает nature-polishing (минуя этот координатор)
  3. Выполняет
```

### Пример 3: Journal club

```
Пользователь: «Journal club по статье Liu et al. 2024, подготовь презентацию и слайды»
Агент:
  1. Загружает scientific-publishing (общий запрос)
  2. Понимает: нужна презентация из статьи → paper2ppt-ru (русская аудитория)
  3. Загружает paper2ppt-ru, делает презентацию
  4. Если нужны bilingual заметки спикера — дополнительно подключает nature-reader
```

## Преимущества координатора

- **Экономия контекста**: этот файл ~6 KB вместо 7 навыков по 13-29 KB каждый (суммарно 150+ KB)
- **Явный mapping**: агент видит таблицу «что→куда» без догадок
- **Расширяемость**: добавить новый Nature-навык = одна строка в таблице

## Связанные навыки (не входят в Nature-cluster, но пересекаются)

- `ai-reads-books` — чтение больших PDF-книг постранично (если статья >50 стр)
- `document-converter` — первичная конвертация PDF → Markdown (перед загрузкой в reader)
- `litprog` — Literate Programming для воспроизводимых вычислений в статье
- `edit-article` — общее редактирование статьи (не научной)

## Антипаттерны

- ❌ Не выполняй работу специалистов самостоятельно — всегда делегируй одному из 7
- ❌ Не загружай все 7 навыков «на всякий случай» — только нужный
- ❌ Не путай `paper2ppt-ru` и `nature-paper2ppt` — первый актуальный, второй legacy
