## 📚 Learning Path Recommendation

### Beginner
1. Math foundations → Python → ML fundamentals
2. Deep Learning Specialization → Practical DL
3. Generative AI → LLM basics

### Intermediate
1. LLM internals → Reasoning LLMs → Multimodal
2. LLM courses (mlabonne, Hugging Face)
3. Frameworks: LangChain, LlamaIndex, Ollama

### Advanced
1. Specializations: CV, NLP, RL
2. Open source LLMs (Llama, Deepseek)
3. Building LLM apps → Agentic coding
