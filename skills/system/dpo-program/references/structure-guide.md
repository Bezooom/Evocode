# Структура рабочей программы ДПО

## Детальная структура

### 1. Пояснительная записка

**1.1. Актуальность программы**
- Анализ состояния предметной области
- Запросы целевой аудитории и работодателей
- Соответствие стратегическим документам
- Научно-практическая значимость

**1.2. Цели и задачи программы**
- Цель: формулируется через глагол действия (сформировать, освоить, освоить)
- Задачи: конкретные шаги достижения цели (3-7 задач)

**1.3. Место программы в структуре ДО**
- Пререквизиты (предварительная подготовка)
- Преемственность с другими программами
- Вклад в формирование компетенций

**1.4. Целевая аудитория**
- Категория слушателей
- Требования к уровню образования
- Предварительные знания и навыки
- Рекомендуемый стаж работы

### 2. Планируемые результаты обучения

**2.1. Компетенции**

| Код | Наименование | Тип |
|---|---|---|
| ПК-X | Профессиональная компетенция X | hard skill |
| ОК-X | Общая компетенция X | soft skill |

**2.2. Результаты по таксономии Блума**

| Результат | Глагол действия | Уровень | Индикатор |
|---|---|---|---|
| Знает | Перечисляет, определяет | 1-2 | ... |
| Умеет | Применяет, анализирует | 3-4 | ... |
| Владеет | Проектирует, оценивает | 5-6 | ... |

### 3. Содержание программы

#### Структура для повышения квалификации (16-72 часа)

```
Раздел 1. [Название] (X часов)
  Тема 1.1. ... (X ч.)
  Тема 1.2. ... (X ч.)
  Практикум 1.1. ... (X ч.)

Раздел 2. [Название] (X часов)
  ...

Итого: XX академических часов
```

#### Структура для переподготовки (250-500 часов)

```
Модуль 1. Базовый (X часов)
  Раздел 1.1. ... (X ч.)
  Раздел 1.2. ... (X ч.)
  Практические занятия: ... (X ч.)
  Самостоятельная работа: ... (X ч.)

Модуль 2. Специальный (X часов)
  ...

Модуль 3. Практико-ориентированный (X часов)
  ...

Итоговая аттестация (X часов)

Итого: XXX академических часов
```

**Каждая тема содержит:**
- Название и объём (акад. часы)
- Вид занятия (лекция, семинар, практика, самостоятельная работа)
- Краткое содержание (2-3 предложения)
- Формируемые компетенции (код)
- Формы контроля

### 4. Фонды оценочных средств

**Для каждого раздела/модуля:**

| № | Тема/раздел | Формат оценки | Инструменты | Критерии |
|---|---|---|---|---|
| 1 | ... | Тест | 20 вопросов | ≥70% |
| 2 | ... | Практическая работа | Кейс-задача | rubrica |
| 3 | ... | Проект | Проектный продукт | rubrica |
| N | Итоговая аттестация | Экзамен | Комплексная задача | rubrica |

**Тестовые задания:**
- Минимум 20 уникальных вариантов для ПК
- Минимум 50 уникальных вариантов для ПП
- Каждый вариант — 15-20 вопросов
- Смешанный уровень сложности

### 6. Кадровое обеспечение

**Требования к преподавателям:**
- Высшее образование (не ниже магистратуры)
- Учёная степень (кандидат/доктор наук) — предпочтительно
- Опыт преподавания — от 3 лет
- Опыт практической работы по профилю — от 5 лет
- Периодическое повышение квалификации

**Эксперты-практики:**
- Действующие специалисты в предметной области
- Руководитель/зам. руководителя (для гос. службы)
- Опыт работы — от 10 лет
- Публикационная активность — предпочтительно
