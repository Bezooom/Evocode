# Коды ОКПД2, ОКЗ, ОКСП для ДПО-программ

## ОКЗ (Общероссийский классификатор занятий)

| Код | Наименование | Примеры программ ДПО |
|---|---|---|
| 1 | Директора, генеральные менеджеры | Стратегическое управление, лидерство |
| 2 | Менеджеры | Управление проектами, HR-менеджмент |
| 3 | Специалисты | IT-специалист, юрист, экономист |
| 4 | Техники | Инженер-программист, техник-строитель |
| 5 | Операторы | Оператор ЭВМ, оператор 1С |

## ОКПД2 (Общероссийский классификатор продукции по видам экономической деятельности)

| Код | Наименование | Примеры |
|---|---|---|
| 85.32.13 | ДПО по программам повышения квалификации | ПК для гос. служащих |
| 85.32.14 | ДПО по программам профессиональной переподготовки | ПП для бюджетников |
| 85.32.19 | ДПО прочее | Тренинги, стажировки |

## Рекомендации по выбору кода

1. **ОКСП** — по основной профильной специальности программы
2. **ОКЗ** — по должности выпускника (для ПП)
3. **ОКПД2** — 85.32.13 для ПК, 85.32.14 для ПП
