# План освоения навыков разработки RTS

Комплексная программа разбита на 5 ключевых модулей, охватывающих все аспекты создания стратегий.

## Модуль 1: UI/UX (Отзывчивость и Информативность)
*   **Теория:** Принципы интерфейсов Blizzard (микро-управление, макро-менеджмент, горячие клавиши, миникарта).
*   **Практика:**
    1.  Реализация точного Raycasting для выбора юнитов (Box Selection, клик по силуэту, двойной клик для выделения типа).
    2.  Создание системы UI, реагирующей на состояния (Sound/Visual feedback при клике, индикация здоровья).
    3.  Разработка функциональной миникарты (отрисовка тумана войны, радар, быстрые прыжки камеры по клику).
    4.  Action Queue: реализация системы очередей команд (выполнение приказов через Shift).

## Модуль 2: Архитектура движка и Масштабируемость
*   **Теория:** Deterministic Lockstep, Entity Component System (ECS), Spatial Partitioning (Quadtrees, Grids).
*   **Практика:**
    1.  Переход на ECS: разделение данных (позиция, здоровье) и систем (перемещение, урон) для обработки 10k+ юнитов.
    2.  Детерминизм: реализация собственной математической библиотеки (Fixed Point Math) для обхода погрешностей float (Floating Point Non-Determinism).
    3.  Продвинутый Pathfinding: комбинация A* для глобальных путей, Flow Fields для групп (swarm/boids) и Local Avoidance (RVO/ORCA) для огибания препятствий.

## Модуль 3: Экономика и Балансировка
*   **Теория:** Кривые добычи ресурсов, темп игры (Pacing), тиры (Tiers) технологий, асимметричный баланс.
*   **Практика:**
    1.  Создание модульной системы характеристик (Modifiers/Buffs).
    2.  Разработка таблиц данных (Excel/JSON) для быстрого тюнинга параметров без изменения кода (Data-Driven Design).
    3.  Прототипирование "петли обратной связи": Сбор -> Постройка -> Армия -> Контроль карты -> Больше ресурсов.
    4.  Инструментарий автотестов для симуляции боев (AI vs AI) с целью выявления сломанных стратегий (Imba).

## Модуль 4: Искусственный Интеллект (ИИ)
*   **Теория:** Finite State Machines (FSM), Behavior Trees (BT), Utility AI, Influence Maps.
*   **Практика:**
    1.  **Микро-уровень:** ИИ юнитов через FSM/BT (поиск укрытия, кайтинг, фокус огня по уязвимым целям).
    2.  **Пространственное мышление:** Создание Influence Maps (Карт влияния) для оценки угрозы на карте и поиска слабых мест в обороне.
    3.  **Макро-уровень:** Создание Директора ИИ (Macro AI), управляющего экономикой, разведкой и формированием волн атак через Utility AI.

## Модуль 5: Визуальный стиль, Анимации и "Juice"
*   **Теория:** Game Feel, 12 принципов анимации Диснея в контексте RTS, Readability.
*   **Практика:**
    1.  Настройка IK (Inverse Kinematics) для адаптации юнитов к ландшафту (правильная постановка ног/гусениц).
    2.  Визуализация обратной связи (Декали под юнитами, партиклы при попадании, тряска камеры при взрывах).
    3.  Реализация тумана войны (Fog of War) со сглаживанием и скрытием объектов вне зоны видимости (Shader-based подход).
    4.  Звуковой ландшафт: Динамическое микширование звуков, чтобы большие скопления юнитов не вызывали аудио-кашу (ограничение одновременно играющих звуков одного типа).
