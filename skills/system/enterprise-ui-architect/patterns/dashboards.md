# Dashboards & Analytics
- Grid: 12-column, responsive breakpoints, widget cards with consistent padding
- KPI Cards: value + delta + sparkline, color-coded status (green/amber/red)
- Charts: accessible legends, data tables fallback, export options
- Real-time: subtle loading indicators, auto-refresh toggle, timestamp
- Empty state: onboarding checklist, "add first widget" CTA
- Sources: Grafana UI, Carbon Charts, Tableau/PowerBI enterprise guidelines
