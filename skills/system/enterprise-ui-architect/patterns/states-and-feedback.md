# States & Feedback
- Loading: skeleton screens (not spinners) for layout stability
- Success: toast + banner, undo option, auto-dismiss (5s)
- Error: clear action ("retry", "contact support"), stack trace hidden, role-aware message
- Confirmation: destructive actions require modal + type-to-confirm or checkbox
- Audit: immutable logs, who/what/when, export trail
- Sources: NN/g, IBM Carbon, Material Enterprise
