# Complex Forms (Enterprise)
- Multi-step: progress indicator, draft saving, validation per step
- Conditional fields: show/hide based on role/data
- Bulk actions: checkboxes, "select all", inline editing
- Validation: inline errors, summary banner, disabled submit until valid
- Layout: 1-2 columns max, labels above inputs, grouped by logical sections
- Sources: IBM Carbon, Ant Design, SLDS
