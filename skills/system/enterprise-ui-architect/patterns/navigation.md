# Enterprise Navigation
- Layout: left sidebar (collapsible) + top bar (user/notifications/search)
- Breadcrumbs: max 4 levels, clickable, dynamic for deep paths
- Workspace/Context switcher: org/project toggle in header
- Search: global command palette (Cmd/Ctrl+K), recent items, filters
- Mobile: drawer pattern, bottom nav for 5 max items, gesture support
- Sources: SLDS Navigation, Carbon UI Shell, Microsoft 365 Hub
