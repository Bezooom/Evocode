## 🎓 Обучение

### Начинающим дизайнерам:
1. Начни с **UX Research** (user-persona, empathy-map)
2. Изучи **Interaction Design** законы (Fitts, Hicks, Miller)
3. Освой **UI Design** основы (color, typography, layout)
4. Практикуй **Prototyping** (wireframes, user-flows)

### Опытным дизайнерам:
1. Внедри **Design Ops** процессы (critique, handoff, sprints)
2. Создай **Design System** (tokens, components, governance)
3. Развивай **UX Strategy** (metrics, service blueprint)
4. Делись знаниями через **Designer Toolkit** (case studies, presentations)

---

## ⚙️ Интеграция в проекты

### Для команд:
- Используй **Design Ops** для организации процессов
- Внедри **Design System** для консистентности
- Проводи **UX Research** перед каждым крупным проектом
- Применяй **Interaction Design** законы для обоснования решений

### Для индивидуальных дизайнеров:
- Создавай **user personas** для каждого проекта
- Используй **design rationale** для объяснения решений
- Собирай **case studies** для портфолио
- Применяй **heuristic evaluation** для самопроверки
