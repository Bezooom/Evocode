# Оформление ссылок и библиографии — ГОСТ Р 7.0.100-2018

## Внутритекстовые ссылки

### Формат

**Простая ссылка на источник:**
> [1]

**Цитирование с указанием страницы:**
> [2, с. 15]

**Цитирование с указанием диапазона страниц:**
> [3, с. 45-50]

**Несколько источников:**
> [1, 3, с. 22] или [1–3]

**Автор + год (авторский стиль):**
> по мнению Иванова (2020), ... / ... (Иванов, 2020)

### Правила

1. Номер ссылки в тексте соответствует номеру в списке литературы
2. Точка ставится ПОСЛЕ скобки со ссылкой: `... как показано в исследовании [5, с. 12].`
3. Ссылка на собственный рисунок/таблицу: `... (рис. 2.1)` или `... (табл. 1.3)`
4. Каждый источник упоминается в списке литературы один раз, независимо от числа ссылок

## Библиографический список

### Общая структура

Список оформляется в соответствии с **ГОСТ Р 7.0.100-2018** «Библиографическая запись. Библиографическое описание. Общие требования и составление».

**Порядок разделов:**
1. Нормативные правовые акты (Конституция, федеральные законы, указы)
2. Международные документы
3. Научная литература (монографии, учебники, статьи)
4. Интернет-ресурсы

**Внутри каждого раздела** — алфавитный порядок.

### Оформление основных типов источников

#### Монография (книга)

**Формат:**
```
Фамилия И. О. Название книги : подзаголовок / И. О. Фамилия авторов. — Издание (если не первое). — Место издания : Издательство, Год. — Количество страниц. — ISBN (если есть).
```

**Примеры:**
```
Болтунин, Н. В. Искусство управления : практическое пособие / Н. В. Болтунин. — 2-е изд., перераб. и доп. — Москва : Питер, 2021. — 320 с. — ISBN 978-5-4461-2085-7.

Smith, J. Strategic Management / J. Smith, M. Jones. — 3rd ed. — London : Routledge, 2020. — 285 p. — ISBN 978-0-367-89456-7.
```

#### Учебное пособие

```
Фамилия И. О. Название : учебное пособие / И. О. Фамилия. — Место : Издательство, Год. — Страницы.
```

**Пример:**
```
Козлов, А. Б. Экономическая теория : учебное пособие / А. Б. Козлов, Е. А. Петрова. — Москва : ИНФРА-М, 2022. — 350 с. — ISBN 978-5-16-108765-4.
```

#### Статья в журнале

```
Фамилия И. О. Название статьи / И. О. Фамилия, И. О. Фамилия // Наименование журнала. — Год. — Том (номер). — С. начало-конец. — DOI (если есть).
```

**Пример:**
```
Иванов, И. И. Методы оптимизации управленческих решений / И. И. Иванов, П. П. Петров // Вопросы экономики. — 2023. — № 5. — С. 45-58. — DOI 10.32686/0042-8736-2023-5-45-58.
```

**Для ВАК/РИНЦ:**
```
Сидоров, С. С. Влияние цифровизации на эффективность бизнеса / С. С. Сидоров // Научный журнал. — 2024. — № 2. — С. 12-20. — DOI: 10.xxxx/nj.2024.2.003. — URL: https://... (дата обращения: 01.03.2024). — Текст : электронный.
```

#### Статья из сборника конференции

```
Фамилия И. О. Название доклада / И. О. Фамилия // Материалы конференции «Название» : сборник трудов. — Место : Издательство, Год. — С. страницы.
```

**Пример:**
```
Козлов, К. К. Анализ рисков в проектной деятельности / К. К. Козлов // Материалы Международной научной конференции «Современные проблемы науки» : сборник трудов. — Москва : МГУ, 2023. — С. 12-15.
```

#### Электронный ресурс

```
Фамилия И. О. Название [Электронный ресурс]. — URL: https://... (дата обращения: DD.MM.YYYY). — Текст : электронный.
```

**Пример:**
```
Федеральный закон от 26.07.2017 № 187-ФЗ «О цифровой экономической среде» [Электронный ресурс]. — URL: http://kremlin.ru/acts/bank/45744 (дата обращения: 15.02.2024). — Текст : электронный.
```

#### Нормативный правовой акт

**Конституция:**
```
Конституция Российской Федерации : принята всенародным голосованием 12.12.1993 (с учётом обновлений, утверждённых Конституционным законом РФ от 30.12.2020 № 1-КК). — Москва : Эксмо, 2021.
```

**Федеральный закон:**
```
Федеральный закон от [дата] № [номер] «[название]» [Электронный ресурс] // Собрание законодательства РФ. — [Год]. — № [номер]. — Ст. [номер статьи]. — URL: http://... (дата обращения: DD.MM.YYYY).
```

**Пример:**
```
Трудовой кодекс Российской Федерации : официальный текст. — Москва : Волтерс Клувер, 2023. — 180 с.
```

#### Диссертация

```
Фамилия И. О. Название диссертации : [тип диссертации : ... степени : ... / № государственной регистрации] / И. О. Фамилия. — Место, Год. — Количество страниц.
```

**Пример:**
```
Смирнова, Е. В. Управленческие инновации в условиях цифровой трансформации : диссертация ... кандидата экономических наук : 08.00.05 / Е. В. Смирнова. — Москва, 2022. — 185 с.
```

#### Иностранные источники (английский)

```
Author, A. A. Title of the book / A. A. Author, B. B. Coauthor. — Edition. — City : Publisher, Year. — Pages. — DOI.
```

**Пример:**
```
Porter, M. E. Competitive Strategy: Techniques for Analyzing Industries and Competitors / M. E. Porter. — New York : Free Press, 2008. — 384 p.

Chen, L. Digital Transformation in Enterprise Management / L. Chen, R. Kumar // Journal of Business Research. — 2023. — Vol. 145. — P. 234-248. — DOI: 10.1016/j.jbusres.2022.03.045.
```

## Автоматизация оформления

Для автоматического оформления списка литературы используйте:

1. **Zotero** (бесплатно) — импорт из eLibrary, Google Scholar, CyberLeninka
2. **BibLaTeX** — для LaTeX-документов
3. **Plugin for Word** — Zotero Word Plugin для вставки ссылок
4. **LegalBook.ru/snoska** — онлайн-генератор ссылок по ГОСТ

## Проверка библиографии

**Чек-лист:**
- [ ] Все источники пронумерованы единообразно
- [ ] Фамилии авторов в алфавитном порядке
- [ ] Годы публикации указаны
- [ ] Страницы статей указаны как «С. 1-10» (не «стр.», не «page»)
- [ ] Интернет-ресурсы имеют дату обращения
- [ ] DOI указан при наличии
- [ ] Не менее 30 источников (бакалавриат), 50 (магистратура)
- [ ] Не менее 30% иностранных источников
- [ ] Не менее 5 источников за последние 3 года
- [ ] Нормативные акты отделены от научной литературы
