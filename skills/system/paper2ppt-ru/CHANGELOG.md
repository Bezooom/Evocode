# Changelog

Все значимые изменения в навыке paper2ppt-ru будут документированы в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-05-12

### Добавлено
- Первая версия навыка paper2ppt-ru
- Автоматическая конвертация научных статей в русскоязычные презентации PPTX
- Поддержка входных форматов: PDF, DOI, arXiv ID, текст статьи
- Определение типа статьи (экспериментальная, обзорная, методологическая, клиническая, теоретическая)
- Интеллектуальный выбор ключевых рисунков
- Генерация структуры слайдов по российским академическим стандартам
- Создание заметок докладчика на русском языке
- Библиографические ссылки по ГОСТ Р 7.0.11-2011
- Специальные шаблоны для защиты диссертаций (кандидатская)
- Поддержка формата 16:9
- Автоматическая верификация качества презентаций
- Извлечение текста и рисунков из PDF через PyMuPDF
- Обработка математических формул (LaTeX → PNG)
- Консервативная цветовая схема для академических презентаций
- Правильное форматирование числовых данных (десятичная запятая, пробелы)
- Шаблоны структур для разных типов статей
- Руководство по стилю для русскоязычных презентаций
- Примеры использования для различных сценариев
- Скрипт верификации презентаций (verify_presentation.py)
- Базовый пример создания презентации (basic_example.py)
- Пример для защиты диссертации (dissertation_example.py)
- Документация по зависимостям (DEPENDENCIES.md)
- Файл requirements.txt с полным списком зависимостей
- Скрипт проверки зависимостей (check_dependencies.py)

### Технические детали
- Минимальная версия Python: 3.8+
- Рекомендуемая версия Python: 3.10+
- Основные зависимости: python-pptx, PyMuPDF, requests, pillow, matplotlib
- Опциональные зависимости: arxiv, biopython, scholarly, pdf2image, sympy, nltk, spacy, pandas, openpyxl

### Документация
- SKILL.md — полное описание навыка (8 шагов workflow)
- README.md — краткое руководство пользователя
- DEPENDENCIES.md — руководство по установке зависимостей
- templates/slide_structures.md — шаблоны структур для разных типов статей
- templates/style_guide.md — руководство по стилю оформления
- examples/EXAMPLES.md — 10 примеров использования
- examples/basic_example.py — базовый пример кода
- examples/dissertation_example.py — пример для защиты диссертации
- examples/verify_presentation.py — скрипт верификации

### Особенности
- Адаптация из nature-paper2ppt (китайский → русский)
- Соответствие ГОСТ Р 7.0.11-2011 для библиографии
- Российские академические стандарты оформления
- Терминология: "актуальность", "научная новизна", "практическая значимость"
- Специальные слайды для защиты диссертаций
- Автоматическая проверка качества (язык, заметки, рисунки, шрифты)

### Поддерживаемые типы презентаций
- Экспериментальная статья (10-12 слайдов)
- Обзорная статья (12-15 слайдов)
- Методологическая статья (10-12 слайдов)
- Клиническая статья (12-15 слайдов)
- Теоретическая статья (10-12 слайдов)
- Защита диссертации (15-18 слайдов)
- Научный семинар (12-16 слайдов)
- Конференция (8-10 слайдов)

### Известные ограничения
- PyMuPDF использует AGPL-3.0 лицензию (требуется коммерческая лицензия для закрытого ПО)
- Извлечение формул из PDF может быть неточным
- Автоматический выбор рисунков может требовать ручной корректировки
- Качество OCR зависит от качества исходного PDF

## Формат версий

Версии следуют формату MAJOR.MINOR.PATCH:
- **MAJOR** — несовместимые изменения API
- **MINOR** — новая функциональность с обратной совместимостью
- **PATCH** — исправления ошибок с обратной совместимостью

## Типы изменений

- **Добавлено** — новая функциональность
- **Изменено** — изменения в существующей функциональности
- **Устарело** — функциональность, которая будет удалена в будущих версиях
- **Удалено** — удаленная функциональность
- **Исправлено** — исправления ошибок
- **Безопасность** — исправления уязвимостей
