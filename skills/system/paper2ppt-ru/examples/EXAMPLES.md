# Примеры использования навыка paper2ppt-ru

## Пример 1: Базовое использование

### Запрос пользователя:
```
Создай презентацию из этой статьи: paper.pdf
```

### Действия агента:

1. **Анализ PDF:**
   - Извлечение текста и метаданных
   - Определение типа статьи: экспериментальная
   - Извлечение 8 рисунков

2. **Определение структуры:**
   - Основной тезис: "Белок X регулирует клеточный цикл через фосфорилирование CDK2"
   - Ключевые доказательства: рисунки 1, 3, 5, 7
   - Структура: Актуальность → Гипотеза → Методы → Результаты → Обсуждение → Выводы

3. **Создание слайдов:**
   - 12 слайдов
   - 4 ключевых рисунка
   - Заметки докладчика для всех слайдов

4. **Результат:**
   ```
   ✅ Презентация создана: presentation_ru.pptx
   ✅ Слайдов: 12
   ✅ Рисунков: 4
   ✅ Заметок докладчика: 12
   ```

## Пример 2: Презентация для защиты диссертации

### Запрос пользователя:
```
Создай презентацию для защиты кандидатской диссертации из моих трех статей:
- article1.pdf (основная работа)
- article2.pdf (методология)
- article3.pdf (клиническое применение)

Специальность: 03.01.03 — Молекулярная биология
```

### Действия агента:

1. **Анализ всех статей:**
   - Извлечение ключевых результатов из каждой статьи
   - Объединение в единую логическую структуру
   - Выделение научной новизны

2. **Создание специальной структуры для диссертации:**
   - Титульный слайд с указанием специальности
   - Актуальность темы диссертации
   - Цель и задачи диссертационного исследования
   - Научная новизна (4 пункта)
   - Теоретическая и практическая значимость
   - Основные результаты по главам
   - Апробация результатов
   - Публикации (3 статьи в ВАК/Scopus)
   - Основные выводы

3. **Результат:**
   ```
   ✅ Презентация для защиты создана: dissertation_presentation_ru.pptx
   ✅ Слайдов: 15
   ✅ Включены все обязательные элементы для защиты
   ✅ Публикации оформлены по ГОСТ Р 7.0.11-2011
   ```

   ✅ Презентация для семинара создана: seminar_presentation_ru.pptx
   ✅ Слайдов: 14

## Советы по использованию

### 1. Уточняй тип презентации
```
Хорошо: "Создай презентацию для защиты диссертации"
Плохо: "Создай презентацию"
```

### 2. Указывай ограничения по времени
```
Хорошо: "Создай презентацию на 10 минут"
Плохо: "Создай презентацию"
```

### 3. Указывай аудиторию
```
Хорошо: "Создай презентацию для студентов (вводный уровень)"
Плохо: "Создай презентацию"
```

### 4. Указывай специфические требования
```
Хорошо: "Создай презентацию с акцентом на клиническое применение"
Плохо: "Создай презентацию"
```

### 5. Проверяй результат
```
После создания:
1. Проверь числовые данные
2. Проверь рисунки
3. Проверь заметки докладчика
4. Прочитай вслух (репетиция)
```

## Часто задаваемые вопросы

**Q: Можно ли создать презентацию из нескольких статей?**
A: Да, просто укажи все файлы: "Создай презентацию из article1.pdf, article2.pdf, article3.pdf"

**Q: Можно ли изменить стиль презентации?**
A: Да, укажи требования: "Создай презентацию с синей цветовой схемой"

**Q: Можно ли добавить дополнительные слайды?**
A: Да, после создания: "Добавь слайд с благодарностями"

**Q: Можно ли изменить порядок слайдов?**
A: Да: "Переставь слайды 5 и 6 местами"

**Q: Можно ли экспортировать в PDF?**
A: Да: "Экспортируй презентацию в PDF"

**Q: Как проверить качество презентации?**
A: Используй скрипт верификации: `python verify_presentation.py presentation_ru.pptx`

## Заключение

Навык paper2ppt-ru позволяет быстро создавать профессиональные русскоязычные презентации из научных статей для любых целей: от защиты диссертации до выступления на конференции.

Ключевые преимущества:
- ✅ Автоматизация рутинной работы
- ✅ Соответствие российским академическим стандартам
- ✅ Библиографические ссылки по ГОСТ
- ✅ Заметки докладчика на русском языке
- ✅ Гибкая настройка под разные типы презентаций
