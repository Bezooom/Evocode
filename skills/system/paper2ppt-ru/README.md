# paper2ppt-ru

Автоматическая конвертация научных статей в русскоязычные презентации PPTX для защиты диссертаций, научных семинаров и конференций.

## Быстрый старт

```bash
# Установка зависимостей
pip install python-pptx PyMuPDF requests pillow matplotlib

# Использование
# Просто скажи агенту: "Создай презентацию из этой статьи"
```

## Возможности

- ✅ Автоматическое извлечение текста и рисунков из PDF
- ✅ Определение типа статьи (экспериментальная, обзорная, методологическая, клиническая, теоретическая)
- ✅ Интеллектуальный выбор ключевых рисунков
- ✅ Генерация структуры слайдов по российским академическим стандартам
- ✅ Создание заметок докладчика на русском языке
- ✅ Библиографические ссылки по ГОСТ Р 7.0.11-2011
- ✅ Специальные шаблоны для защиты диссертаций
- ✅ Поддержка формата 16:9
- ✅ Автоматическая верификация качества

## Входные форматы

- PDF файл (локальный или URL)
- DOI (автоматическая загрузка)
- arXiv ID
- Текст статьи
- HTML от издателя

## Выходные файлы

```
paper2ppt_output/
├── presentation_ru.pptx          # Готовая презентация
├── figures/                       # Извлеченные рисунки
├── metadata.json                  # Метаданные
├── slide_structure.md             # Структура слайдов
└── verification_report.txt        # Отчет о проверке
```

## Типы презентаций

### Экспериментальная статья
- Актуальность → Гипотеза → Методы → Результаты → Обсуждение → Выводы
- 10-15 слайдов
- Фокус на ключевых экспериментальных данных

### Обзорная статья
- Проблема → Систематизация → Анализ → Тренды → Выводы
- 12-18 слайдов
- Классификация и сравнительный анализ

### Методологическая статья
- Проблема → Новый метод → Валидация → Сравнение → Применение
- 10-14 слайдов
- Алгоритмы и схемы

### Клиническая статья
- Клиническая проблема → Дизайн → Результаты → Клиническое значение
- 12-16 слайдов
- Характеристики пациентов, эффективность, безопасность

### Теоретическая статья
- Проблема → Модель → Математика → Предсказания → Верификация
- 10-15 слайдов
- Формулы и теоретические выкладки

## Специальные возможности

### Для защиты диссертации
Автоматически добавляются слайды:
- Научная новизна
- Теоретическая значимость
- Практическая значимость
- Апробация результатов
- Публикации по теме диссертации

### Для конференции
- Краткий формат (10-15 слайдов)
- Фокус на ключевых результатах
- Контактная информация

### Для научного семинара
- Подробные методы
- Обсуждение ограничений
- Перспективы развития

## Стандарты оформления

- **Формат:** 16:9
- **Шрифт:** Arial, Calibri или PT Sans
- **Размер шрифта:** заголовок 28-32pt, текст 20-24pt
- **Цветовая схема:** консервативная (синий, серый, белый)
- **Числовые данные:** десятичная запятая, пробел между числом и единицей
- **Библиография:** ГОСТ Р 7.0.11-2011

## Требования к качеству

- ✅ Минимум 8 слайдов
- ✅ Максимум 25 слайдов (для 20-минутного доклада)
- ✅ Все слайды на русском языке
- ✅ 80% слайдов имеют заметки докладчика
- ✅ Информативные заголовки
- ✅ Читаемые рисунки (минимум 300 DPI)
- ✅ Логическая последовательность

## Troubleshooting

### Рисунки не извлекаются
Используй альтернативный метод через pdf2image:
```bash
pip install pdf2image
```

### Формулы нечитаемы
Увеличь DPI до 600 при сохранении формул.

### Текст не помещается
Автоматическое уменьшение шрифта для длинных текстов.
