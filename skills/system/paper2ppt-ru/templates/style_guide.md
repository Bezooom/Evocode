# Руководство по стилю для русскоязычных презентаций

## Общие принципы

### 1. Язык и терминология

**Русский язык:**
- Все слайды на русском языке
- Заметки докладчика на русском языке
- Подписи к рисункам на русском языке

**Терминология:**
- Используй принятые русские термины
- Англицизмы в скобках при первом упоминании
- Пример: "машинное обучение (machine learning, ML)"
- Избегай калек с английского

**Примеры правильной терминологии:**
```
✅ ПРАВИЛЬНО:
- машинное обучение (machine learning)
- нейронная сеть (neural network)
- глубокое обучение (deep learning)
- обработка естественного языка (natural language processing, NLP)

❌ НЕПРАВИЛЬНО:
- машинный лёрнинг
- нейронка
- дип лёрнинг
- НЛП (без расшифровки)
```

### 2. Числовые данные

**Десятичный разделитель:**
- Используй запятую, а не точку
- Пример: 3,14 (не 3.14)
- Пример: 0,05 (не 0.05)

**Единицы измерения:**
- Пробел между числом и единицей: 25 °C, 100 мл, 5 кг
- Исключения: проценты без пробела: 95%
- Диапазоны через тире: 10–20 мкм, 5–10 мг

**Большие числа:**
- Разделяй пробелами: 1 000 000 (не 1,000,000)
- Или используй степени: 10⁶

**Примеры:**
```
✅ ПРАВИЛЬНО:
- Температура: 37,5 °C
- Концентрация: 10 мкМ
- Диапазон: 5–10 нм
- Процент: 95%
- Большое число: 1 000 000 или 10⁶

❌ НЕПРАВИЛЬНО:
- Температура: 37.5°C
- Концентрация: 10мкМ
- Диапазон: 5-10 нм
- Процент: 95 %
- Большое число: 1,000,000
```

### 3. Типографика

**Шрифты:**
- Основной текст: Arial, Calibri или PT Sans
- Размер заголовка: 28-32pt
- Размер текста: 20-24pt
- Минимальный размер: 18pt

**Начертание:**
- Заголовки: полужирный
- Основной текст: обычный
- Акценты: курсив или полужирный (умеренно)

**Выравнивание:**
- Заголовки: по центру или по левому краю
- Текст: по левому краю
- Числа в таблицах: по правому краю

### 4. Цветовая схема

**Консервативная палитра:**
```
Основные цвета:
- Фон: белый (#FFFFFF) или светло-серый (#F5F5F5)
- Текст: темно-серый (#333333) или черный (#000000)
- Акценты: синий (#0066CC), темно-синий (#003366)

Дополнительные цвета:
- Успех/положительное: зеленый (#2E7D32)
- Предупреждение: оранжевый (#F57C00)
- Ошибка/отрицательное: красный (#C62828)
- Нейтральное: серый (#757575)
```

**Избегай:**
- Ярких неоновых цветов
- Низкого контраста (светлый текст на светлом фоне)
- Более 4 цветов на одном слайде

### 5. Структура слайда

**Заголовок:**
- Информативный, а не декоративный
- Отражает суть слайда
- Пример: ❌ "Результаты" → ✅ "Метод X повышает точность на 25%"

**Контент:**
- Максимум 6-7 строк текста
- Маркированные списки (не сплошной текст)
- Один ключевой месседж на слайд

**Пример хорошего слайда:**
```
Заголовок: Экспрессия белка X увеличивается в 3 раза

Контент:
• Контрольная группа: 100 ± 15 у.е.
• Экспериментальная группа: 300 ± 25 у.е.
• Статистическая значимость: p < 0,001
• Подтверждено методами ИФА и Вестерн-блот

[Рисунок: график с планками погрешностей]
```

### 6. Рисунки и графики

**Требования к рисункам:**
- Разрешение: минимум 300 DPI
- Формат: PNG (с прозрачностью) или JPEG
- Размер: занимает 50-70% слайда
- Подписи: на русском языке, под рисунком

**Подписи к рисункам:**
```
Формат: Рисунок [номер]. [Описание]

Примеры:
✅ Рисунок 1. Иммунофлуоресцентное окрашивание белка X в клетках HeLa
✅ Рисунок 2. Зависимость активности фермента от концентрации субстрата
✅ Рисунок 3. Схема экспериментального дизайна

❌ Fig. 1. Immunofluorescence staining
❌ Рис. 1
❌ Окрашивание
```

**Графики:**
- Оси подписаны на русском
- Единицы измерения указаны
- Легенда читаема
- Планки погрешностей показаны

### 7. Таблицы

**Структура таблицы:**
- Заголовок над таблицей
- Шапка таблицы выделена
- Выравнивание: текст влево, числа вправо
- Чередующиеся строки (для больших таблиц)

**Пример:**
```
Таблица 1. Сравнение методов

| Метод    | Точность, % | Время, с | Память, МБ |
|----------|-------------|----------|------------|
| Метод A  |      87     |    1,2   |     150    |
| Метод B  |      92     |    2,5   |     200    |
| Метод C  |      95     |    1,8   |     180    |
```

### 8. Формулы

**Отображение формул:**
- Крупный размер (минимум 20pt)
- Центрирование
- Нумерация справа (если нужно)
- Пояснение переменных

**Пример:**
```
E = mc²                                                    (1)

где:
  E — энергия, Дж
  m — масса, кг
  c — скорость света, м/с
```

### 9. Библиографические ссылки

**Формат по ГОСТ Р 7.0.11-2011:**

**Статья в журнале:**
```
Иванов И.И., Петров П.П. Название статьи // Название журнала. 
Год. Т. XX. № X. С. XX–XX.

Пример:
Иванов И.И., Петров П.П. Роль белка X в регуляции клеточного цикла // 
Молекулярная биология. 2024. Т. 58. № 2. С. 234–245.
```

**Книга:**
```
Автор А.А. Название книги. Город: Издательство, Год. XXX с.

Пример:
Иванов И.И. Молекулярная биология клетки. М.: Наука, 2023. 450 с.
```

**Иностранная статья:**
```
Author A.A., Author B.B. Article title // Journal Name. Year. Vol. XX. 
No. X. P. XX–XX.

Пример:
Smith J., Brown A. Protein X regulates cell cycle // Nature. 2024. 
Vol. 625. P. 123–135.
```

### 10. Заметки докладчика

**Структура заметок:**
- Полные предложения (не тезисы)
- Связки между слайдами
- Пояснения к сложным терминам
- Ответы на возможные вопросы
- Тайминг: ~2-3 минуты на слайд

**Пример заметок:**
```
Слайд: Результат 1: Экспрессия белка X увеличивается в 3 раза

Заметки докладчика:
"На этом слайде представлены результаты количественного анализа 
экспрессии белка X. Как видно на графике, в экспериментальной группе 
уровень экспрессии составил 300 условных единиц, что в три раза выше, 
чем в контрольной группе, где он составил 100 условных единиц. 

Различия статистически значимы с уровнем p меньше 0,001, что было 
подтверждено с помощью t-критерия Стьюдента. Эти результаты были 
получены двумя независимыми методами: иммуноферментным анализом и 
Вестерн-блоттингом, что повышает надежность наших выводов.

Таким образом, мы можем утверждать, что белок X действительно играет 
ключевую роль в исследуемом процессе. Переходим к следующему результату."
```

### 11. Специфика для защиты диссертации

**Обязательные элементы:**
- Специальность (шифр и название)
- Актуальность темы диссертации
- Научная новизна (3-4 пункта)
- Теоретическая значимость
- Практическая значимость
- Апробация результатов
- Публикации по теме диссертации

**Формулировки:**
```
✅ ПРАВИЛЬНО:
- "Научная новизна работы заключается в..."
- "Теоретическая значимость состоит в..."
- "Практическая значимость определяется..."
- "Апробация результатов проведена на..."

❌ ИЗБЕГАЙ:
- "Мы сделали..."
- "Я исследовал..."
- Неформальных выражений
```

### 12. Анимация и переходы

**Рекомендации:**
- Минимум анимации (или без нее)
- Простые переходы между слайдами
- Избегай эффектов "вылета", "вращения" и т.п.
- Для академических презентаций: без анимации

### 13. Формат презентации

**Соотношение сторон:**
- Предпочтительно: 16:9
- Допустимо: 4:3 (для старых проекторов)

**Размер файла:**
- Оптимизируй изображения
- Целевой размер: до 20 МБ
- Максимум: 50 МБ

### 14. Контрольный чек-лист

**Перед финализацией проверь:**
- [ ] Все слайды на русском языке
- [ ] Десятичные запятые (не точки)
- [ ] Пробелы между числами и единицами
- [ ] Размер шрифта минимум 18pt
- [ ] Информативные заголовки
- [ ] Не более 7 строк текста на слайд
- [ ] Рисунки высокого разрешения
- [ ] Подписи к рисункам на русском
- [ ] Библиографические ссылки по ГОСТ
- [ ] Заметки докладчика для 80% слайдов
- [ ] Формат 16:9
- [ ] Консервативная цветовая схема
- [ ] Нет орфографических ошибок
- [ ] Нет опечаток в числах

### 15. Типичные ошибки

**Избегай:**
```
❌ Слишком много текста на слайде
❌ Мелкий шрифт (меньше 18pt)
❌ Низкий контраст (серый текст на белом фоне)
❌ Декоративные заголовки ("Введение", "Результаты")
❌ Смешение русского и английского на одном слайде
❌ Десятичные точки вместо запятых
❌ Отсутствие единиц измерения
❌ Нечитаемые рисунки
❌ Отсутствие заметок докладчика
❌ Яркие неоновые цвета
❌ Избыточная анимация
❌ Неправильные библиографические ссылки
```

### 16. Примеры хороших и плохих слайдов

**Плохой слайд:**
```
Заголовок: Результаты

Текст:
В ходе проведенного нами исследования было установлено, что экспрессия 
белка X в экспериментальной группе значительно превышает таковую в 
контрольной группе. Данные были получены методом иммуноферментного 
анализа и подтверждены методом Вестерн-блоттинга. Статистический анализ 
показал высокую степень достоверности различий между группами.

Проблемы:
- Неинформативный заголовок
- Слишком много текста
- Сплошной текст вместо списка
- Нет конкретных цифр
```

**Хороший слайд:**
```
Заголовок: Экспрессия белка X увеличивается в 3 раза

Контент:
• Контроль: 100 ± 15 у.е.
• Эксперимент: 300 ± 25 у.е.
• Различия: p < 0,001
• Методы: ИФА, Вестерн-блот

[График с планками погрешностей]

Преимущества:
- Информативный заголовок
- Краткие тезисы
- Конкретные цифры
- Визуализация
```

---

## Заключение

Следование этим правилам стиля обеспечит создание профессиональных, академически корректных русскоязычных презентаций, соответствующих российским стандартам научных докладов.
