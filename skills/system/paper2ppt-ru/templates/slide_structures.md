# Шаблоны структур презентаций

## Экспериментальная статья

```yaml
type: experimental
slides:
  - title: "Титульный слайд"
    content:
      - Название статьи
      - Авторы
      - Журнал, год, DOI
    
  - title: "Актуальность исследования"
    content:
      - Контекст проблемы
      - Нерешенные вопросы
      - Научная новизна
    notes: "Объяснить, почему эта проблема важна"
    
  - title: "Гипотеза исследования"
    content:
      - Основная гипотеза
      - Предсказания
    notes: "Четко сформулировать проверяемую гипотезу"
    
  - title: "Цель и задачи"
    content:
      - Цель исследования
      - Задачи (3-4 пункта)
    notes: "Связать задачи с гипотезой"
    
  - title: "Материалы и методы"
    content:
      - Объект исследования
      - Экспериментальная модель
      - Основные методы
      - Статистический анализ
    notes: "Кратко описать экспериментальный дизайн"
    
  - title: "Результат 1: [Конкретное утверждение]"
    content:
      - Рисунок/график
      - Ключевые данные
    notes: "Детально объяснить результат и его интерпретацию"
    
  - title: "Результат 2: [Конкретное утверждение]"
    content:
      - Рисунок/график
      - Ключевые данные
    notes: "Связать с предыдущим результатом"
    
  - title: "Результат 3: [Конкретное утверждение]"
    content:
      - Рисунок/график
      - Ключевые данные
    notes: "Показать, как результаты подтверждают гипотезу"
    
  - title: "Обсуждение результатов"
    content:
      - Интерпретация результатов
      - Сравнение с литературой
      - Механизм (если применимо)
      - Ограничения исследования
    notes: "Критически оценить результаты"
    
  - title: "Выводы"
    content:
      - 3-5 конкретных выводов
      - Практическая значимость
      - Перспективы
    notes: "Подчеркнуть вклад в науку"
    
  - title: "Благодарности"
    content:
      - Финансирование
      - Соавторы
      - Контакты
    notes: ""

estimated_duration: "15-20 минут"
recommended_slides: "10-12"
```

## Обзорная статья

```yaml
type: review
slides:
  - title: "Титульный слайд"
    content:
      - Название обзора
      - Авторы
      - Журнал, год
    
  - title: "Проблема и область обзора"
    content:
      - Актуальная проблема
      - Границы обзора
      - Цель обзора
    notes: "Объяснить, почему нужен этот обзор"
    
  - title: "Методология обзора"
    content:
      - Базы данных
      - Критерии включения/исключения
      - Временной период
      - Количество источников
    notes: "Показать систематичность подхода"
    
  - title: "Классификация подходов"
    content:
      - Схема или таблица классификации
      - Основные категории
    notes: "Объяснить логику классификации"
    
  - title: "Подход 1: [Название]"
    content:
      - Описание подхода
      - Преимущества
      - Недостатки
      - Примеры применения
    notes: "Детально разобрать первый подход"
    
  - title: "Подход 2: [Название]"
    content:
      - Описание подхода
      - Преимущества
      - Недостатки
      - Примеры применения
    notes: "Сравнить со вторым подходом"
    
  - title: "Сравнительный анализ"
    content:
      - Таблица сравнения
      - Критерии оценки
      - Рейтинг подходов
    notes: "Объективно сравнить подходы"
    
  - title: "Выявленные пробелы"
    content:
      - Нерешенные проблемы
      - Противоречия в литературе
      - Недостаточно изученные аспекты
    notes: "Показать, что еще нужно исследовать"
    
  - title: "Тренды и перспективы"
    content:
      - Текущие тренды
      - Перспективные направления
      - Прогнозы развития
    notes: "Дать видение будущего области"
    
  - title: "Выводы"
    content:
      - Основные выводы обзора
      - Рекомендации
    notes: "Суммировать ключевые моменты"

estimated_duration: "20-25 минут"
recommended_slides: "12-15"
```

## Методологическая статья

```yaml
type: methodological
slides:
  - title: "Титульный слайд"
    content:
      - Название метода
      - Авторы
      - Журнал, год
    
  - title: "Проблема существующих методов"
    content:
      - Ограничения метода A
      - Ограничения метода B
      - Необходимость нового подхода
    notes: "Мотивировать разработку нового метода"
    
  - title: "Описание нового метода"
    content:
      - Основная идея
      - Алгоритм (блок-схема)
      - Ключевые особенности
    notes: "Объяснить суть метода простым языком"
    
  - title: "Математическая основа"
    content:
      - Основные формулы
      - Теоретическое обоснование
    notes: "Дать математическое обоснование"
    
  - title: "Реализация метода"
    content:
      - Программная реализация
      - Вычислительная сложность
      - Требования к ресурсам
    notes: "Показать практическую реализуемость"
    
  - title: "Валидация на тестовых данных"
    content:
      - Тестовые наборы данных
      - Метрики качества
      - Результаты валидации
    notes: "Показать, что метод работает"
    
  - title: "Сравнение с аналогами"
    content:
      - Таблица сравнения
      - Метрики: точность, скорость, память
      - Статистическая значимость
    notes: "Объективно сравнить с конкурентами"
    
  - title: "Примеры применения"
    content:
      - Пример 1: [область]
      - Пример 2: [область]
      - Результаты применения
    notes: "Показать универсальность метода"
    
  - title: "Ограничения и перспективы"
    content:
      - Текущие ограничения
      - Направления улучшения
      - Планы развития
    notes: "Честно обсудить ограничения"
    
  - title: "Доступность"
    content:
      - Открытый код (GitHub)
      - Документация
      - Примеры использования
    notes: "Показать, как другие могут использовать метод"
    
  - title: "Выводы"
    content:
      - Преимущества нового метода
      - Области применения
      - Вклад в область
    notes: "Подчеркнуть значимость метода"

estimated_duration: "15-20 минут"
recommended_slides: "10-12"
```

    content:
      - Основные выводы

      - Перспективы
    notes: "Подчеркнуть клиническую значимость"

estimated_duration: "20-25 минут"
recommended_slides: "12-15"

## Примечания по использованию шаблонов

1. **Выбор шаблона**: Определите тип статьи и выберите соответствующий шаблон
2. **Адаптация**: Адаптируйте структуру под конкретную статью
3. **Заголовки**: Делайте заголовки информативными, а не декоративными
4. **Заметки**: Всегда добавляйте заметки докладчика
5. **Время**: Рассчитывайте ~2 минуты на слайд
6. **Рисунки**: Включайте только ключевые рисунки
7. **Текст**: Минимум текста, максимум визуализации
