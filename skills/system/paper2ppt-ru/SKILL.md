# Конвертация научных статей в русскоязычные презентации PPTX

Конвертация научных статей в русскоязычные презентации PPTX для защиты диссертаций, научных семинаров и конференций.

---

## Purpose

Превратить научную статью в **полную, эффективную, академически корректную** русскоязычную PPTX-презентацию, готовую для защиты диссертации, научного семинара, конференции или лабораторной встречи.

**Не используй** этот навык для:
- Кратких резюме или bullet-списков ключевых слов
- Задач только с цитированием
- Презентаций без научного контента

---

## Core Principle

> **Полнота + Эффективность + Академическая корректность**

1. **Полнота**: Покрывает всю логику статьи — от актуальности до выводов
2. **Эффективность**: Выбирает только необходимые рисунки, избегает избыточности
3. **Академическая корректность**: Следует российским стандартам научных презентаций
4. **Русский язык**: Все слайды, заметки докладчика и подписи на русском языке
5. **ГОСТ-совместимость**: Библиографические ссылки по ГОСТ Р 7.0.11-2011

## Workflow

### Шаг 1: Импорт и анализ статьи

**Входные форматы:**
- PDF файл (локальный путь или URL)
- DOI (автоматическая загрузка через Unpaywall/Sci-Hub)
- arXiv ID (например, `2401.12345`)
- Текст статьи (скопированный пользователем)
- HTML от издателя

**Действия:**
```python
# Если PDF
import fitz  # PyMuPDF
doc = fitz.open("paper.pdf")
full_text = ""
for page in doc:
    full_text += page.get_text()

# Если DOI
import requests
doi_url = f"https://api.unpaywall.org/v2/{doi}?email=user@example.com"
response = requests.get(doi_url)
pdf_url = response.json()['best_oa_location']['url_for_pdf']

# Если arXiv
arxiv_pdf = f"https://arxiv.org/pdf/{arxiv_id}.pdf"
```

**Извлечь:**
- Заголовок, авторы, журнал, год
- Аннотация
- Введение (актуальность, цель)
- Методы
- Результаты
- Обсуждение
- Выводы
- Список рисунков и таблиц
- Библиографические ссылки

---

### Шаг 2: Определение типа статьи и структуры аргументации

**Типы статей:**

| Тип | Структура презентации |
|-----|----------------------|
| **Экспериментальная** | Актуальность → Гипотеза → Методы → Результаты → Обсуждение → Выводы |
| **Обзорная** | Проблема → Систематизация → Анализ → Тренды → Выводы |
| **Методологическая** | Проблема существующих методов → Новый метод → Валидация → Сравнение → Применение |
| **Клиническая** | Клиническая проблема → Дизайн исследования → Пациенты → Результаты → Клиническое значение |
| **Теоретическая** | Проблема → Теоретическая модель → Математический аппарат → Предсказания → Верификация |

**Определи:**
1. Тип статьи
2. Основной тезис (главное утверждение)
3. Ключевые доказательства (какие рисунки/таблицы критичны)
4. Логическую цепочку аргументации

---

### Шаг 3: Выбор рисунков и таблиц

**Правило эффективности:**
> Включай только рисунки/таблицы, которые **необходимы** для понимания основного тезиса.

**Критерии отбора:**
- ✅ Ключевые результаты, подтверждающие гипотезу
- ✅ Схемы экспериментального дизайна
- ✅ Графики с основными трендами
- ✅ Сравнительные таблицы (метод vs метод)
- ❌ Дублирующие данные
- ❌ Вспомогательные материалы (Supplementary)
- ❌ Избыточные статистические таблицы

**Пример:**
```
Статья: 8 рисунков
Презентация: 3-4 ключевых рисунка
```

### Шаг 4: Создание структуры слайдов

**Стандартная структура для российских академических презентаций:**

```
1. Титульный слайд
   - Название статьи
   - Авторы
   - Журнал, год, DOI
   - Докладчик (если отличается от авторов)

2. Актуальность и проблема
   - Контекст исследования
   - Нерешенная проблема
   - Научная новизна

3. Цель и задачи
   - Основная цель
   - Конкретные задачи (2-4)

4. Материалы и методы
   - Объект исследования
   - Методы исследования
   - Схема эксперимента (если есть)

5-N. Результаты (по ключевым рисункам)
   - Один слайд = один ключевой результат
   - Рисунок + краткая интерпретация

N+1. Обсуждение
   - Интерпретация результатов
   - Сравнение с литературой
   - Ограничения исследования

N+2. Выводы
   - 3-5 конкретных выводов
   - Практическая значимость

N+3. Благодарности (опционально)
   - Финансирование
   - Соавторы
```

### Шаг 5: Написание контента слайдов

**Принципы русскоязычного академического стиля:**

1. **Заголовки слайдов:**
   - Информативные, не декоративные
   - Пример: ❌ "Результаты" → ✅ "Метод X повышает точность на 25%"

2. **Текст на слайдах:**
   - Краткие тезисы (не полные предложения)
   - Маркированные списки
   - Максимум 6-7 строк на слайд
   - Размер шрифта: заголовок 28-32pt, текст 20-24pt

3. **Терминология:**
   - Используй принятые русские термины
   - Англицизмы в скобках при первом упоминании
   - Пример: "машинное обучение (machine learning, ML)"

4. **Числовые данные:**
   - Десятичная запятая (не точка): 3,14 (не 3.14)
   - Пробел между числом и единицей: 25 °C, 100 мл
   - Диапазоны через тире: 10–20 мкм

5. **Библиографические ссылки:**
   - По ГОСТ Р 7.0.11-2011
   - Пример: "Иванов И.И., Петров П.П. Название статьи // Журнал. 2023. Т. 10. № 2. С. 15–23."

---

### Шаг 6: Создание заметок докладчика

**Для каждого слайда напиши заметки докладчика на русском языке:**

```
Слайд 3: Цель и задачи

Заметки докладчика:
"Основная цель нашего исследования — разработать новый метод X для решения проблемы Y.

Для достижения этой цели мы поставили четыре задачи:
Первая — провести систематический анализ существующих подходов.
Вторая — разработать математическую модель процесса.
Третья — экспериментально валидировать модель на реальных данных.
Четвертая — сравнить наш метод с современными аналогами.

Переходим к методам исследования."
```

**Требования к заметкам:**
- Полные предложения (не тезисы)
- Связки между слайдами
- Пояснения к сложным терминам
- Ответы на возможные вопросы
- Тайминг: ~2-3 минуты на слайд

### Шаг 7: Генерация PPTX файла

**Используй python-pptx:**

```python
from pptx import Presentation
from pptx.util import Inches, Pt
from pptx.enum.text import PP_ALIGN

# Создать презентацию
prs = Presentation()
prs.slide_width = Inches(10)  # 16:9 формат
prs.slide_height = Inches(5.625)

# Титульный слайд
title_slide_layout = prs.slide_layouts[0]
slide = prs.slides.add_slide(title_slide_layout)
title = slide.shapes.title
subtitle = slide.placeholders[1]

title.text = "Название статьи на русском языке"
subtitle.text = "Авторы: Иванов И.И., Петров П.П.\nЖурнал Nature, 2024\nDOI: 10.1038/xxxxx"

# Слайд с контентом
content_slide_layout = prs.slide_layouts[1]
slide = prs.slides.add_slide(content_slide_layout)
title = slide.shapes.title
content = slide.placeholders[1]

title.text = "Актуальность исследования"
tf = content.text_frame
tf.text = "Проблема X остается нерешенной"
p = tf.add_paragraph()
p.text = "Существующие методы имеют ограничения"
p.level = 1

# Добавить рисунок
img_path = 'figure1.png'
left = Inches(1)
top = Inches(2)
height = Inches(4)
pic = slide.shapes.add_picture(img_path, left, top, height=height)

# Заметки докладчика
notes_slide = slide.notes_slide
text_frame = notes_slide.notes_text_frame
text_frame.text = "Заметки докладчика на русском языке..."

# Сохранить
prs.save('presentation_ru.pptx')
```

**Стиль оформления:**
- Шрифт: Arial, Calibri или PT Sans (для кириллицы)
- Цветовая схема: консервативная (синий, серый, белый)
- Фон: белый или светло-серый
- Избегай ярких цветов и анимаций

### Шаг 8: Верификация и QA

**Автоматические проверки:**

```python
# 1. Проверка структуры
def verify_structure(prs):
    assert len(prs.slides) >= 5, "Минимум 5 слайдов"
    assert prs.slides[0].shapes.title.text, "Титульный слайд должен иметь заголовок"
    
# 2. Проверка языка
def verify_language(prs):
    for slide in prs.slides:
        for shape in slide.shapes:
            if hasattr(shape, "text"):
                # Проверка на кириллицу
                cyrillic_count = sum(1 for c in shape.text if '\u0400' <= c <= '\u04FF')
                assert cyrillic_count > 0, "Текст должен быть на русском"

# 3. Проверка заметок докладчика
def verify_notes(prs):
    slides_with_notes = 0
    for slide in prs.slides:
        if slide.notes_slide.notes_text_frame.text.strip():
            slides_with_notes += 1
    assert slides_with_notes >= len(prs.slides) * 0.8, "80% слайдов должны иметь заметки"

# 4. Проверка рисунков
def verify_figures(prs):
    figure_count = 0
    for slide in prs.slides:
        for shape in slide.shapes:
            if shape.shape_type == 13:  # Picture
                figure_count += 1
    assert figure_count >= 2, "Минимум 2 рисунка"
```

**Ручная проверка:**
- [ ] Все термины переведены корректно
- [ ] Числовые данные соответствуют статье
- [ ] Рисунки читаемы и подписаны
- [ ] Логическая связность между слайдами
- [ ] Заметки докладчика полные и связные
- [ ] Библиографические ссылки по ГОСТ

## Paper-Type Guidance

### Экспериментальная статья

**Фокус:** Гипотеза → Эксперимент → Результаты → Интерпретация

**Ключевые слайды:**
1. Актуальность и гипотеза
2. Экспериментальный дизайн (схема)
3. Основные результаты (2-3 слайда с графиками)
4. Статистический анализ
5. Интерпретация и выводы

**Пример заголовков:**
- "Гипотеза: Белок X регулирует процесс Y"
- "Экспериментальная модель: клеточная линия HeLa"
- "Результат 1: Экспрессия белка X увеличивается в 3 раза"

---

### Обзорная статья

**Фокус:** Систематизация → Анализ → Тренды → Перспективы

**Ключевые слайды:**
1. Проблема и область обзора
2. Методология поиска литературы
3. Классификация подходов (таблица или схема)
4. Сравнительный анализ
5. Выявленные пробелы
6. Перспективные направления

**Пример заголовков:**
- "Обзор охватывает 150 публикаций за 2015-2024 гг."
- "Классификация методов машинного обучения для задачи X"
- "Пробел: отсутствие методов для случая Y"

---

### Методологическая статья

**Фокус:** Проблема существующих методов → Новый метод → Валидация

**Ключевые слайды:**
1. Ограничения существующих методов
2. Описание нового метода (алгоритм, схема)
3. Валидация на тестовых данных
4. Сравнение с аналогами (таблица)
5. Примеры применения
6. Доступность (код, данные)

**Пример заголовков:**
- "Проблема: метод A не работает для больших данных"
- "Новый алгоритм: O(n log n) вместо O(n²)"
- "Точность: 95% vs 87% у лучшего аналога"

---

### Клиническая статья

**Фокус:** Клиническая проблема → Исследование → Клиническое значение

**Ключевые слайды:**
1. Клиническая проблема и актуальность
2. Дизайн исследования (когорта, критерии)
3. Характеристики пациентов (таблица)
4. Основные результаты (выживаемость, эффективность)
5. Безопасность и побочные эффекты
6. Клинические рекомендации

**Пример заголовков:**
- "Проблема: 30% пациентов не отвечают на терапию X"
- "Когорта: 250 пациентов, рандомизированное исследование"
- "Результат: медиана выживаемости увеличилась на 8 месяцев"

## Style Rules

### Текст на слайдах

```
✅ ХОРОШО:
Актуальность исследования
• Проблема X затрагивает 30% населения
• Существующие методы имеют точность 70%
• Необходим новый подход

❌ ПЛОХО:
Актуальность
В настоящее время проблема X является актуальной, так как она затрагивает 
значительную часть населения. Существующие методы, несмотря на их широкое 
применение, демонстрируют недостаточную точность...
```

### Рисунки

```python
# Извлечение рисунков из PDF
import fitz
doc = fitz.open("paper.pdf")
for page_num in range(len(doc)):
    page = doc[page_num]
    images = page.get_images()
    for img_index, img in enumerate(images):
        xref = img[0]
        base_image = doc.extract_image(xref)
        image_bytes = base_image["image"]
        with open(f"figure_{page_num}_{img_index}.png", "wb") as f:
            f.write(image_bytes)
```

**Требования к рисункам:**
- Разрешение: минимум 300 DPI
- Формат: PNG (с прозрачностью) или JPEG
- Подписи: на русском языке, под рисунком
- Размер: занимает 50-70% слайда

### Таблицы

```python
from pptx.util import Inches

# Данные
table.cell(1, 0).text = "Метод A"
table.cell(1, 1).text = "87"
table.cell(1, 2).text = "1.2"
```

## Quality Rules

### Обязательные требования

1. **Язык:**
   - ✅ Все слайды на русском языке
   - ✅ Заметки докладчика на русском языке
   - ✅ Подписи к рисункам на русском языке
   - ❌ Не смешивай русский и английский на одном слайде

2. **Структура:**
   - ✅ Минимум 8 слайдов
   - ✅ Максимум 25 слайдов (для 20-минутного доклада)
   - ✅ Логическая последовательность
   - ✅ Титульный слайд + слайд с выводами

3. **Контент:**
   - ✅ Каждый слайд имеет информативный заголовок
   - ✅ Текст краткий (не более 7 строк)
   - ✅ Рисунки читаемы и релевантны
   - ✅ Числовые данные соответствуют статье

4. **Заметки докладчика:**
   - ✅ Минимум 80% слайдов имеют заметки
   - ✅ Заметки полные (2-3 минуты речи)
   - ✅ Связки между слайдами
   - ✅ Пояснения к сложным терминам

5. **Оформление:**
   - ✅ Единый стиль шрифтов
   - ✅ Консервативная цветовая схема
   - ✅ Формат 16:9
   - ✅ Читаемый размер шрифта (минимум 20pt)

### Рекомендации

1. **Для защиты диссертации:**
   - Добавь слайд "Научная новизна"
   - Добавь слайд "Практическая значимость"
   - Добавь слайд "Апробация результатов"
   - Укажи публикации и конференции

## Troubleshooting

### Проблема: Рисунки не извлекаются из PDF

**Решение:**
```python
# Используй альтернативный метод
from pdf2image import convert_from_path

images = convert_from_path('paper.pdf', dpi=300)
for i, image in enumerate(images):
    image.save(f'page_{i}.png', 'PNG')
```

### Проблема: Формулы нечитаемы

**Решение:**
```python
# Увеличь DPI для формул
plt.savefig('formula.png', dpi=600, bbox_inches='tight')
```

### Проблема: Текст не помещается на слайд

**Решение:**
```python
# Автоматическое уменьшение шрифта
from pptx.util import Pt

for paragraph in text_frame.paragraphs:
    for run in paragraph.runs:
        if len(text_frame.text) > 300:
            run.font.size = Pt(18)
        else:
            run.font.size = Pt(22)
```
