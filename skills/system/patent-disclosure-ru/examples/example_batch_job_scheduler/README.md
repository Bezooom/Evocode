# Пример: распределённое планирование пакетных задач и восприятие ресурсов

Этот каталог — **вымышленный учебный проект**, содержит только **`knowledge/`** как **материалы для сканирования Шаг 2** (документация, примеры Office, фрагменты кода). Патентные решения, чек-лист, заявка — генерировать в **`outputs/{идентификатор_проекта}/`** (см. [examples/README.md](../README.md) в корне).
