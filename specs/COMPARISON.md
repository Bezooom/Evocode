## Детальное сравнение

### Локальная модель

**Эвокод:** Встроенный inference-фреймворк класса llama.cpp, скомпилированный в дистрибутив. Поддержка двух моделей (FIM + Chat) с shared memory. Пользователь подгружает GGUF-модели.

**Cursor:** Использует llama.cpp через встроенный сервер. Поддержка нескольких моделей (Llama, Claude, GPT). Менее гибкая настройка.

**GitHub Copilot:** Работает только через облако (GitHub servers). Не поддерживает локальный вывод.

**Windsurf:** Встроенный локальный сервер на основе Codeium. Поддержка нескольких моделей.

**Codeium:** Только облачный вывод. Локальная работа ограничена.

### Автономная работа

**Эвокод:** Полностью автономный — работает без интернета, все данные остаются на устройстве.

**Cursor:** Частично автономный — локальный вывод работает офлайн, но синхронизация требует интернета.

**GitHub Copilot:** Только онлайн — все данные отправляются в облако GitHub.

**Windsurf:** Частично автономный — локальный вывод работает офлайн.

**Codeium:** Только онлайн — все данные отправляются в облако.

### Гибридный роутинг

**Эвокод:** Умный AI Router с эвристическим скорером сложности. Автоматическое переключение между локальным и облачным выводом на основе `complexity_score`.

**Cursor:** Есть переключение между моделями, но менее интеллектуальное.

**GitHub Copilot:** Только облако.

**Windsurf:** Есть переключение между моделями.

**Codeium:** Только облако.

### Самоактуализация

**Эвокод:** Skill Sync Engine — ежедневная синхронизация навыков из GitHub. Автоматическое обновление, diff, rollback. Поддержка оверрайдов.

**Cursor:** Есть автообновления, но без разделения на системные/пользовательские каталоги.

**GitHub Copilot:** Только ручное обновление навыков.

**Windsurf:** Есть автообновления.

**Codeium:** Только ручное обновление.

### Приватность

**Эвокод:** Локальный вывод не покидает устройство. DLP Guard маскирует секреты перед отправкой в OpenRouter.

**Cursor:** Локальный вывод остаётся на устройстве, но синхронизация требует интернета.

**GitHub Copilot:** Все данные отправляются в облако GitHub.

**Windsurf:** Локальный вывод остаётся на устройстве.

**Codeium:** Локальный вывод остаётся на устройстве.
